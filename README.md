# Fin_cooling_MSSD

This project implements a finite element solver for transient heat conduction in a two-dimensional copper cooling fin using FEniCSx (DOLFINx).

## Features

- Mesh generation using Gmsh
- Transient heat conduction solved with the Backward Euler method
- Convective and Dirichlet boundary conditions
- Verification using the Method of Manufactured Solutions (MMS)
- Mesh convergence study
- Comparison of natural and forced convection cooling

## Requirements

- Python 3.11+
- DOLFINx
- Gmsh
- PETSc
- NumPy
- Matplotlib

## Output

- Temperature field visualizations
- Mesh convergence results
- Heat dissipation histories
- Figures used in the final report
